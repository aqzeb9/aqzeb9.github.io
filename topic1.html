Binary System

The binary system is the foundation of all computing. It uses only two digits, 0 and 1, to represent data. Every piece of information in a computer is stored using binary code.

Binary works because computers use electrical signals. A signal that is on represents 1, and a signal that is off represents 0.

Advantages of Binary

  • Simple and efficient
  • Reliable for electronic systems
  • Easy to process

Decimal Binary
1 0001
5 0101
